# Claudo — Book Club App

A web app for running virtual and in-person book clubs. Track your current read, manage members, schedule meetings, and keep discussion notes all in one place.

## Features (planned)

- **Clubs** — create and join book clubs, virtual or in-person
- **Current book** — set the active book with cover, author, and reading schedule
- **Meetings** — schedule sessions with video link or location
- **Discussion** — post questions and notes before/after meetings
- **Members** — invite readers via email or link

## Tech stack

- Frontend: React + TypeScript + Vite
- Backend: Node.js + Express (coming soon)
- Database: SQLite (local dev) → PostgreSQL (production)

## Getting started

```bash
npm install
npm run dev
```
